## Technologies
This project was developed with the following:
- Node.js
- React Native
- Sqlite3
- REST API

<br>

## Project
Proffy is an online study platform that helps people find teachers online.

<br>

## Getting Started
- Install node.js
- Run npm install
- To run frontend: yarn Start
- To run backend: yarn dev
